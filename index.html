这是一份中文的参会指南，会议详情参见大会官网。

W3C 2023年度全球技术大会（TPAC 2023）将于今年九月召开：

  • 时间：9月11-15日
  • 地点：西班牙塞维利亚（支持远程参会）
  • 注册：注册须知与注册入口（截至9月2日）

提示：我们了解到，由于各种原因并非每个人都能够负担 TPAC 注册费，但我们不希望这些成为大家参与 TPAC 的顾虑。因此，我们向远程与现场参会者均提供注册费用减免政策（不限制人数）。费用减免将建立在信任的基础上。如有需要，与会者可在会议注册页面勾选”注册费减免“的选项。

大会简介：W3C Technical Plenary / Advisory Committee Meetings Week（简称 TPAC）是 W3C 一年一度的全球技术大会。届时，W3C 董事会（Board of Directors）、技术架构组（TAG）、咨询委员会（AB）、会员代表组成的顾问委员会（AC）、W3C 各技术小组、特邀专家以及全球社区成员，将通过为期一周的集中互动交流，深入探讨未来开放 Web 平台技术方向。

TPAC 由工作小组会议、自由话题分组讨论、会员交流会、开发者大会等多个环节构成。其中：

  • 小组会议: W3C 数十个工作组、兴趣组、商务组、社区组集中召开小组会议及小组间的联合会议。小组会议对小组成员、会员单位成员、特邀嘉宾开放，讨论规范制定及小组工作进展与计划。多个小组将于会前发布年度工作汇报视频。
  • 分组话题讨论：全球与会者自由分组话题讨论近年来广受欢迎，该环节将汇集全体与会者，以自由分组讨论的形式围绕当下数十个技术热点话题及标准化需求与趋势展开广泛交流。任何会员及公众均可自行提议希望得到广泛探讨的话题。本年度分组话题提议现已开始，详见参与指南。
  • 会员交流会：AC 会议对全球会员单位代表开放，讨论 W3C 组织策略、联盟架构、标准流程及未来技术方向。
  • 开发者大会：免费面向公众开放，全球 Web 开发人员、应用开发者、产品及项目经理将在此积极分享交流 Web 前沿技术
